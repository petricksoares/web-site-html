#  Portfólio Pessoal- HTML
Este projeto consiste no desenvolvimento de um site de portfólio pessoal utilizando apenas HTML, criado como parte de uma atividade acadêmica da disciplina de Desenvolvimento Web. 
O objetivo principal é aplicar os conceitos fundamentais da linguagem HTML, com foco em estrutura semântica, organização de conteúdo e boas práticas de marcação.

## Objetivos do Projeto
* Aplicar os conceitos básicos e avançados de HTML.
* Utilizar corretamente tags semânticas.
* Criar uma estrutura organizada e de fácil leitura.
* Desenvolver um site simples, funcional e acessível.

## Estrutura do Site
Cabeçalho (Header): Nome, título profissional e foto.

Menu de Navegação (Nav): Links internos para facilitar a navegação.

Sobre Mim: Breve apresentação pessoal e objetivo profissional.

Formação Acadêmica: Informações sobre a formação.

Habilidades Técnicas: Tabela com principais competências.

Áreas de Interesse: Principais áreas de atuação desejadas.

Localização: Mapa incorporado via iframe.

Rodapé (Footer): Informações finais do projeto.

## 🛠️ Tecnologias Utilizadas
* HTML5
* VSCODE
